Dit is de repository voor de leer roblox site. Leer roblox is een site waar je gratis kunt leren games maken in roblox.


# setup voor ontwikelaars
Als je zelf dingen wilt aanpassen op de site moet je node js op je computer hebben geinstaleerd. Als je dat gedaan hebt clone je deze repository. Daarna open je een terminal in de folder van de gekloonde repository. En run de volgende command:
```
npm install
```
als dat gedaan is dan kan je de site starten met:
```
npm start
```
De site werkt op docusaurus. lees zeker hu ndocumentatie is een keer: https://docusaurus.io/docs
