Nu dat roblox studio geinstaleerd is kunnen we ons eerste spel aanmaken. Wanneer je roblox studio opent kom je op de volgende pagina terecht:

![roblox studio start](/rondleiding/studio-start.png)

Hier krijg je verschilende opties. De meeste van deze opties zijn games die al op voorand zijn gemaakt. Dit worden tempaltes genoemt. Maar wij willen een nieuw spel maken dus druk op het vakje waar staat: **Classic Baseplate**

:::tip
Je kan later ook is een keertje 1 van de voorgemaakt games proberen om te kijken hoe deze in elkaar zitten.
:::

Nu word je naarje nieuw spel gebracht. Als je studio er nu ongeveer zoals dit uitziet is alles goed gelukt en kan je anar het volgende stuk gaan.
![roblox studio start](/rondleiding/nieuw-project.png)
