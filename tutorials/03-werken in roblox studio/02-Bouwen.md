Als je een nieuw project maakt in roblox dan begin je meestal met een lege wereld. We kunnen veel verschilenden dingen toevoegen aan deze wereld. Maar meestal ga je in je wereld werken met "Parts". Er zijn veel verschilenden soorten parts, de meest voorkomende is een block. Maar parts kunnen ook in de vorm van een ball, cilinder of wigblok.
![Parts](/tutorials/rondleiding/parts.png)

## Een part toevoegen
Het aanmaken van een part is heel simpel. Druk vanboven in studio op de knop "Part". Als je op deze knop drukt zal er blok worden toegevoegd in je wereld. Wil je een ander vorm toevoegen dan kan je onder de knop op het pijltje drukken en een andere vorm kiezen. 
![Part knop](/tutorials/rondleiding/part-knop.png)
<img src={require('@site/assets/tutorials/rondleiding/nieuw-part.png').default} width="300" />

:::caution
Als je een andere vorm kiest dan zal wanneer je op de gewone knop drukt de laatst gekozen vorm geplaatst worden. Om er voor te zorgen dat deze knop terug een blok plaats kies je bij de vormen gewoon een blok.
:::

## De 4 bewerk tools
Er zijn 4 tools om een part te bewerken:

![4 tools](/tutorials/rondleiding/4-tools.png)

- **De select tool**: met deze tool kunnen we parts in de viewport selecteren.
- **De move tool**: word gebruikt voor het verplaatsen van parts in de viewport.
- **De Scale tool**: word gebruikt voor het vergrote/verkleinen van parts in de viewport.
- **De rotate tool**: word gebruikt voor het draaien van parts in de viewport.

### Select tool
Met de select tool kan je dingen die in de viewport staan selecteren. Het is ook mogelijk om dingen te verplaatsen met de select tool door linker muisknop in te houden op het object dat je wilt verplaatsen, en dan kan je het object slepen naar waar je wilt.

### Move tool
Een beter manier om dingen te verplaatsen is de move tool. Met deze tool kan je objecten zeer specifiek verplaatsen. Als je op de tool klikt en dan een object in de viewport zullen er 6 pijlen verschijnen. 1 pijl voor elke richting. Deze pijlen kan je slepen om het object in de richting te verplaatsen
<video width="320" height="240" controls>
  <source src={require('@site/assets/tutorials/rondleiding/move-tool.mp4').default} type="video/mp4"/>
</video>

### Scale tool
De scale tool word gebruikt om objecten te vergrootten. Als je een object selecteerde met de scale tool verschijnen er 6 bollen. Met deze bollen kan je een object vergrootten of verkleinen in een bepaalde richting. 
<video width="320" height="240" controls>
  <source src={require('@site/assets/tutorials/rondleiding/scale-tool.mp4').default} type="video/mp4"/>
</video>

### Rotate tool
Met de rotate tool kan je objecten draaien. klik op een object dat je wilt draaien. rond dit object zullen 3 cirkels verschijnen. Met deze cirkels kan je het object doen rond draaien.
